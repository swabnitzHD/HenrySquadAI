## Overview

Henry Squad AI is an intelligent chat application that leverages AI to provide seamless conversation experiences. The project features a beautifully designed homepage and modern, responsive user interface built with TypeScript and CSS.

- 💬 **AI-Powered Chat** - Intelligent conversation engine

- 📱 **Responsive UI** - Works seamlessly across all devices
- ⚡ **Fast Performance** - Optimized for quick loading and interactions
- 🔒 **Type Safe** - Built with TypeScript for reliability

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/swabnitzHD/HenrySquadAI.git

# Navigate to the project directory
cd HenrySquadAI

# Install dependencies
npm install
# or
yarn install
```

### Running the Application

```bash
# Start the development server
npm start
# or
yarn start
```

The application will be available at `http://localhost:3000` (or your configured port).

### Building for Production

```bash
# Build the project
npm run build
# or
yarn build
```
